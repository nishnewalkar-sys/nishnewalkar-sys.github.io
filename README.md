## 🚀 Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/nishnewalkar-sys/nishnewalkar-sys.github.io.git
   ```

2. **Local Testing**
   - Open `index.html` in a web browser
   - Or use a local server:
   ```bash
   python -m http.server 8000
   ```
   - Visit `http://localhost:8000`

3. **Deploy to GitHub Pages**
   - The website is automatically deployed to: `https://nishnewalkar-sys.github.io`
   - No additional setup needed!

## ⚙️ Customization Guide

### Update Contact Information

- Email: `nisha@nishacounsellor.in`

- Location: `Thane, Mumbai`

### Modify Colors
Edit `:root` variables in `styles.css`:
```css
--primary-color: #2c5f7f;
--secondary-color: #4a90a4;
--accent-color: #7cb9d9;
```

### Add Blog Posts
1. Create new HTML files in the `blog/` directory
2. Update blog links in `index.html`

### Add Resources
1. Place PDF files in a `resources/` directory
2. Update download links in the Resources section
